# ⚡ VYRON AI: Advanced System Capabilities & Features

VYRON AI is a Visionary Reasoning Operational Neural Interface designed for kernel-level system automation, data analysis, and advanced workflow orchestration. Here is the complete list of system capabilities:

---

## 1. Core Neural Voice Engine
* **High-Speed WebSocket Array**: Real-time bidirectional voice chat with sub-1.5s response latency.
* **Smart Model Routing**: Dynamically directs instructions between Groq (for instant processing) and Gemini (for vision and complex context operations).
* **Cross-Session Memory**: Locally cached context that allows VYRON to remember details across sessions.
* **Global System Hotkeys**: Toggle listening status immediately from any background app using hardware hook triggers (`Ctrl+Alt+V`).

---

## 2. Creative Suite Integration
Hands-free workflow automation using keyboard-shortcut injection models:
* **Adobe Premiere Pro**: Split, cut, trim, paste, insert media assets, and navigate timelines dynamically.
* **Adobe Photoshop**: Instant active tool selection (V-move, B-brush, Pen, Lasso, Crop) and quick layer adjustments.
* **Adobe After Effects**: Property reveals (Position, Scale, Opacity, Rotation) and composition triggers.
* **Canva Web**: Automated navigation controls and project sharing links.

---

## 3. Office & Analytics Expert
AI-powered document generation and spreadsheet automation:
* **PowerPoint Deck Builder**: Generates dark-themed presentation slides (`.pptx`) with custom text structures, visual layouts, slow-fade transitions, and relevant downloaded images.
* **Excel Data Writer**: Bulk range writes, automatic index generation, serial number formatting (e.g. `1-10` fills), and formulas.
* **Tally Prime & Power BI**: Keystroke automation macros for ledger vouchers, balance sheets, and report refreshes.
* **Custom Animated Visualizations**: Generates floating CustomTkinter charts displaying progress indicators, performance analytics, or custom bar graphs.

---

## 4. Live PDF Viewer & Visual Editor
* **Interactive PDF Reader**: Loads local PDFs in a custom scrollable window (`CTkToplevel`) with highlighted keyword searching.
* **Rename & File Processing**: Voice-triggered document renaming and structure organization.
* **Find-and-Replace Text Edit**: Scans PDF text streams and makes custom string edits dynamically.

---

## 5. Security & System Gates
* **Facial Verification (BossCam)**: Uses local camera feeds to authenticate user profile before executing critical commands.
* **Secure Vault**: Access restrictions guarded by custom 4-digit security PIN layers.
* **One-Time Key License System**: Restricts app access to verified purchase keys (`VYRON-XXXXX`) checked against a serverless Next.js API route.

---

## 6. System & Disk Operations
* **PowerShell Routines**: Empties Recycle Bin, wipes temporary folders, and cleans active caches on command.
* **System Utilities**: Hands-free control over master volume levels, screens, brightness, and running system tasks.
